# Государственный пожарный контроль и надзор

Веб-портал для комплексного управления пожарной безопасностью:
журнал выездов, государственный учёт пожаров, модуль государственного
контроля и надзора, CRM и документооборот.

## Технологии

- Node.js / TypeScript
- Vite + React (клиент)
- PostgreSQL (Neon) + Drizzle ORM
- Развёртывание и разработка в Replit

## Назначение

Внутренняя система для:
- ведения журналов выездов и отчётности (1-ОСП, 2-ССГ и др.);
- учёта подконтрольных объектов и проверок;
- ведения документов, версий и аналитики.

Репозиторий используется для дальнейшей доработки системы с помощью ИИ-агентов Replit (Jules).
